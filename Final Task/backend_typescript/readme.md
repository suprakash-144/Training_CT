# Task Management Application (Full Stack)

A full stack application for team, project and task management developed using MERN stack.

## Dependencies

- Backend
  - Typescript
  - express
  - Mongodb
  - joi
  - jsonwentokens
  - swagger

## Project Setup

- Backend
  - Run command `npm run build` to build and deploy the server
